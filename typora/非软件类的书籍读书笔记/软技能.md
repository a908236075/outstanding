# 第一篇 职业

1. 转变为工资打工的心态,而是为公司服务,服务的对象越多,才能获取更多的回报.
2. 营销自己,将自己树立成一个品牌.
3. 获取工作的最好的方式是获得内推的机会.
4. 选择一个方向,作为自己的特长.例如 web开发,分布式,中间件等.
5. 不同规模公司的对比
   1. ![](..\picture\软技能\不同规模公司的对比.png)
6. 晋升方式是承担更多的责任,然后让领导知道,成为问题的解决者.
7. 如果想创业需要从第二职业开始.
8. 不要恐惧未来,不要选择逃避,相信事情终会有解决的办法,自信起来.
9. 拥抱技术的日新月异,拥抱变化,不拘泥于自己已经掌握的知识.

# 第二篇 自我营销

1. 意识到自我营销的重要性:例如同样优秀的乐队,有的是全国演唱会有的只能在酒吧.
2. 创建自己的品牌,有人说咖啡就想到星巴克一样.
3. 博客是推销自己即廉价又简单的方式.热门博客在创作的时候也是不可预知的,所以不要怕自己的幼稚.
4. 给他人想要的东西,把自己的成果的90%做成免费的.

# 第三篇 学习

1. 学习最终的目的是自己的实践,并能够教会其它的人.
2. 如果没有时间系统的学习,需要意识到二八定律了,即学会20%的方法就会应对平时80%的工作,其它都是非常细节的东西,不用在第一时间掌握.
3. **十步学习法**(1-6只做一次,7-10重复做)
   1. **了解全局**:维基百科,官网,大致浏览书籍的目录来了解所要学习知识的总体.
   2. **确定范围**:向一小部分聚焦,非常容易犯的错误就是试图解决太大的问题而把自己搞的不堪重负.聚焦到一个主题,抵制住扩张学习范围的诱惑.
   3. **定义目标**:学习后最终要达成的效果,清晰的定义这个效果,使它可以检验.例如一个基本功能的实现.
   4. **寻找资源**:信息的来源不要仅仅限于书籍,博客,官网,视频都是可以的.
   5. **创建学习计划**:规划步骤,一步一步循序渐进的执行.
   6. **筛选资源**:精简资料,避免对重复的知识复出过多的时间.
   7. **开始学习浅尝辄止**:只学基础的知识,经受住想要学习全貌的诱惑.
   8. **动手操作边玩边学**:实践非常的重要
   9. **全面掌握学以致用**:有了基本知识,不全其余的20%
   10. **教授别人**

# 第四篇 生产力

1. 保持专注:做事情有时候不想做就不去做,我们应该倒过来,让让身体先进入到状态,然后倒逼脑子.先尝试5到10分钟看看,会发现渐入佳境的.
2. 番茄工作法
   1. 可以确认一周内有多少个番茄钟.
   2. 制定每日的计划,并不能够控制一项任务完成需要花费的时间,有时即使非常忙也会完不成任务,即使已经完成了所有的任务,放松的时候还是会觉得有罪恶感.
   3. 我们不一定能够控制完成一项任务到底需要多少时间，但是我们可以控制自己这一天中愿意为某项任务（或某些任务）花多少时间.
   4. 理论上8小时是16个番茄钟,实际坐着每天的目标是10个番茄钟.一周是50个番茄钟.
3. 不要多任务并行处理,如果想并行,就选择一个不费脑子和需要精神专注的任务组合起来.可以一边健身一边听有声读物.
4. 培养惯性,例如每天早起看书,坚持足够长就会形成新的习惯.
5. 坏的习惯容易让自己拖延,例如每天看邮件,浏览网页.
6. 学会分解任务:大任务会使人不知从何处下手,产生未知的胆怯.

# 第七篇 精神

1. 如果不相信自己成功,那么确实很难成功.
2. 积极思考有益于身体健康.
3. 身体需要锻炼,同样思想也是需要锻炼的,积极的看问题没有想象的那么简单.
4. 一直告诉你的大脑,你的理想形象是什么,并且坚信你一定可以变成那样.让你的大脑也这么认为.
5. 即使在冷门的知识也是有人喜欢的,尽管认为现在的自己不怎么完美,但是同样有人喜欢的,所以不要怕.
6. 失败不等于被打败,失败往往是自己进步最快的时刻,所以正视失败,不要害怕它.
